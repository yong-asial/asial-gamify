# Asial Corporation — Website Overview

Source: <https://www.asial.co.jp/> · Captured: 2026-06-12

## Company Overview

Asial is a Tokyo-based technology company founded in **2002**, led by Representative
Director **Masahiro Tanaka (田中正裕)**. It brands itself as a *"Digital Empowerment
Company"* — an expert group using IT to make people and society more prosperous,
covering everything from backend systems to frontend and UI/UX design.

## Why Choose Us (選ばれる理由)

A full-stack specialist team: members possess the complete skill set from requirements
definition through UI/UX design, allowing complex client challenges to be addressed in a
single engagement and high-value proprietary products to be developed internally.

## Solutions (8)

1. **System Consulting** — 20+ years of development expertise across industries
2. **DX / In-house Development Support** — hands-on technical guidance to promote DX
3. **System Construction & App Development** — web and mobile apps with modern tech
4. **AI System Development & R&D** — data analysis, PoC, and AI-integrated UI/UX
5. **UX/UI Design** — multi-platform design, designer + developer collaboration
6. **University DX Support** — DX solutions for higher-education operations
7. **Information Education** — promoting technology literacy since founding (2002)
8. **Products** — Monaca, Onsen UI, and other development tools

## Products (7)

- **Monaca** — cloud-based hybrid mobile app dev platform (web standards)
- **Monaca Education** — programming education for schools (JS/Python, build real apps)
- **Monaca for Hybridcast** — HTML5 content tooling for next-gen "Hybridcast" TV
- **Onsen UI** — open-source mobile UI framework (100+ components, React/Vue)
- **MyCampus** — university DX app (digital student ID, attendance, campus services)
- **ObjectiveFS** — S3-backed unlimited, NFS-compatible cloud storage
- **JpGraph** — PHP graphing library (20+ chart types)
